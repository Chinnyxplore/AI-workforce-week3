The Difference between Copliot Chat, Copilot for Business(M365 copilot) and Copilot Studio
Copilot Chat; is basically used by the end user which is a free version which comes by default on windows 11. It is basically use for creating, summarizing and analyzing work using natural language.
It results is provided based on what is was asked, It is gotten from the web. It is basically used for "Gossiping"
Everyone has access to it because it does not need a License to be used. It can be easily accessed without a License.
Copilot For Business(Microsoft365 Copilot); Is the Copilot used with a License this means for one to have access to this Copilot one has to have a Microsoft 365 Business or Enterprise account.
It Provide AI powered conversation using the web Data with limited work context and also limited integration of Microsoft365 Organization.
Microsoft365 copilot is integrated into Microsoft365 Application which means the information provided is gotten from the Data accessed from the Application like; Outlook, Teams, word, Ecxel, Onenote etc.
It uses Organiztional data security. It requires apprpriate licensing and correction permission.
Copilot Studio; This also requires a paid subscription. It is used to build copilot agents or workflows. 
It is basically used by the Administratorts, Power user or solution builders.
Generally speaking, Copilot Studio is used in birthing Artificial Intelligent Agents.
This is done with the use of the LLMS(Large Language Model) which is the engine behinde building the agents and also Microsft Graph.
This provide the data and context while LLM inerprets propmts and generates naturalhuman like responses.
